Welcome to the MBARI Media Managment (M3) Server

This server is running NGINX. It is also acting as a proxy for the REST services hosted on a number of MBARI servers. The source code for the NGINX docker build is located at https://bitbucket.org/mbari/m3-deployspace/src/master/nginx/

M3 Services

Raziel

Raziel provides service configuration information, such as microservice URLs and access credentials, to VARS Applications. If needed, use http://[this server name]/config as the configuration URL in VARS Annotation and VARS Gridview.
  • Swagger docs
  • Service Health
  • Health of all M3 services

Annosaurus

Annosaurus stores and retrieves annotation information. The default endpoint is http://[this server name/anno
  • Swagger docs
  • Service Health
  • Find a few imaged moments

Vampire-squid

Vampire-squid is a video asset manager that organizes videos by dive or deployment. The default endpoint is http://[this server name]/vam
  • Swagger docs
  • Service Health
  • Show all deployment IDs (aka video sequence names)

VARS Knowedgebase Server

The VARS Knowledgebase server provides access to the taxonomy used by MBARI's annotation systems. The default endpoint is http://[this server name]/kb
  • Service Health
  • Show phylogeny branch down to the genus Nanomia

VARS User Server

The VARS User server manages user accounts and user preferences. The default endpoint is http://[this server name]/users
  • Service Health
  • Show all users

Panoptes

Panoptes archives framegrabs from annotation applications

Beholder
